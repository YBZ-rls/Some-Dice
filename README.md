# Some-Dice

Console level dice-roller application  

Allows you to roll up to 9 (planned) dice of different sizes at the same time  

**Supported dice (planned)**  
- d2 (coin)  
- d4 (pyramid)  
- d6 (cube)  
- d8 (okay so I got tired of naming the geometric shapes)
- d10  
- d12  
- d20  
- d100
